# Debugging on Android Emulator

## Running App on Emulator

The app is being run on the Android emulator with verbose logging to capture all errors.

## What We're Checking

1. **App Launch** - Does it open or crash immediately?
2. **Firebase Initialization** - Any Firebase errors?
3. **Route Errors** - Any navigation issues?
4. **Widget Errors** - Any UI rendering problems?
5. **Permission Issues** - Any missing permissions?

## Logs Being Captured

- Flutter framework errors
- Dart exceptions
- Firebase initialization errors
- Route navigation errors
- Widget build errors

## Expected Output

If app works:
- Emulator window opens
- App installs
- Login screen appears

If app crashes:
- Error messages in terminal
- Stack traces
- Specific error locations
